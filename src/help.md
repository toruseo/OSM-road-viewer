# OSM Road Viewer 操作方法

## 基本操作

### 地図の操作
- **移動**: マウスドラッグ
- **ズーム**: マウスホイール
- **回転**: Ctrl + マウスドラッグ
- **傾斜**: Ctrl + マウスドラッグ（上下）

### 道路情報の確認
- 道路にマウスを置くと、道路名がツールチップで表示されます

## 検索機能

左上の検索パネルで道路を検索できます。

- **道路名**: 部分一致で検索（例: 「東名」）
- **種別**: プルダウンから選択
- **路線番号**: 完全一致で検索（例: 「1」）

複数の条件を指定するとAND検索になります。
空欄の項目は無視されます。

## 表示オプション

- **道路名を表示**: チェックを入れると道路名ラベルが表示されます

## 道路の色分け

| 種別 | 色 |
|------|-----|
| motorway（高速道路） | 赤 |
| trunk（国道等） | 青 |
| primary（主要地方道） | 緑 |
| secondary（一般県道） | 緑（細） |
| その他 | グレー |
