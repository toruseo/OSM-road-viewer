# OSM-road-viewer

**WIP**

OSM道路データをWebGLで高速描画するビューア

## 機能
- 道路種別（fclass）による色分け表示
- 道路名ラベル表示（オン/オフ切替）
- ホバー時ツールチップ
- 道路検索（name, fclass, ref）

## セットアップ
```bash
npm install
```

## 開発
```bash
npm run dev
```

## ビルド
```bash
npm run build
```
